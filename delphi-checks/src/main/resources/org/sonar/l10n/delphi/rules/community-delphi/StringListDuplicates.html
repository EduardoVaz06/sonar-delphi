TStringLists must be sorted if setting the Duplicates property.
For the purposes of this rule, the list must be sorted on the next or previous line.

See:
- System.Classes.TStringList.Duplicates 

    
List1.Duplicates := dupIgnore; // Noncompliant

List2.Duplicates := dupError; // Noncompliant

List3.Duplicates := dupAccept; // Compliant (dupAccept doesn't require a sorted list)

List4.Sorted := True;
List4.Duplicates := dupIgnore; // Compliant

List5.Sorted := True;
List5.Duplicates := dupError; // Compliant
