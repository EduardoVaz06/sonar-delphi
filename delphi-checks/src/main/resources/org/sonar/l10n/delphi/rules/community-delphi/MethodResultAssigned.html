In functions, Result must be assigned a value.

Any parameter with an out specifier must also be assigned a value.
For more information, see the Embarcadero wiki page for Out Parameters 

    
function Test: String;
begin
  // Noncompliant
end;

function Test: TArray;
begin
  // Noncompliant
end;

function Test: String;
begin
  Result := ''; // Compliant
end;

function Test: TArray;
begin
  Result := TArray.Create(); // Compliant
end;
