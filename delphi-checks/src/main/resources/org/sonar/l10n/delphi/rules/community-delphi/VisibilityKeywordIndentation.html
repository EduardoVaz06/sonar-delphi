Visibility specifier directives (private, public, etc.) should use the same indentation as the type name that they are declared under. They should not be indented in the same way as a field, method, or property.

Please note:
- This rule checks for an exact match on trailing whitespace - mixed indentation may yield unexpected results.

For more information, see the Class and Record Declarations section of the Embarcadero style guide page on Type Declarations
