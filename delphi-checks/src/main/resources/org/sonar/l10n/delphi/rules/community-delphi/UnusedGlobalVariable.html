Unused global variables should be removed.
Leaving them in reduces the code's readability, since their presence can be confusing.

Exclusions:
Form variables generated by the IDE are excluded from this rule.
The exclusion is based on the following heuristic:
- A variable declaration section is the last element in the interface section
- There is a single variable declaration in the variable declaration section.
- The variable declaration's type descends from TComponent
